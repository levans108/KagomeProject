# Exotic Charge Ordering in the Kagome Lattice

This repository contains the Mathematica code used to generate the numerical results for the MSci project **“Exotic Charge Ordering in the Kagome Lattice”**.

## Contents

## Repository contents

- `Kagome2.nb` — main notebook containing the iterative solver, critical interaction calculations, and phase diagram generation  
- `Kagome dispersion relation + band structure.nb` — notebook used to derive and visualise the dispersion relation and band structure

## Description

The code implements a nearest-neighbour tight-binding and mean-field model for CDW ordering on the kagome lattice. Self-consistent ordering parameters are obtained numerically, and the resulting solutions are used to construct phase diagrams in $(T,V,\mu)$ space. The band structure and dispersion relation are found from the eigenvalues of the unit cell hamiltonian.

## Requirements

- Wolfram Mathematica

## Notes

Some phase diagram calculations are computationally intensive and may take several hours depending on the chosen resolution.
